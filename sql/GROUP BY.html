GROUP BY ​

GROUP BY 是SQL中用于分组聚合的关键字，它将数据按照指定的列进行分组，然后对每个分组进行聚合计算。

GROUP BY 的作用 ​

GROUP BY 将多行数据合并成一行，通常配合聚合函数使用。

基本语法 ​

sql
SELECT column1, aggregate_function(column2)
FROM table
GROUP BY column1;

常用聚合函数 ​

sql
-- 统计每个部门的员工数量
SELECT department, COUNT(*) as employee_count
FROM employees 
GROUP BY department;

-- 计算每个部门的平均年龄
SELECT department, AVG(age) as avg_age
FROM employees 
GROUP BY department;

-- 找出每个部门的最高和最低工资
SELECT department, MAX(salary) as max_salary, MIN(salary) as min_salary
FROM employees 
GROUP BY department;

-- 计算每个部门的总工资
SELECT department, SUM(salary) as total_salary
FROM employees 
GROUP BY department;

多列分组 ​

sql
-- 按部门和性别分组统计
SELECT department, gender, COUNT(*) as count
FROM employees 
GROUP BY department, gender;

实际应用场景 ​

1. 电商订单分析 ​

sql
-- 统计每个客户的订单数量和总金额
SELECT 
    customer_id,
    COUNT(*) as order_count,
    SUM(amount) as total_amount,
    AVG(amount) as avg_amount
FROM orders 
GROUP BY customer_id;

2. 销售数据分析 ​

sql
-- 按月统计销售额
SELECT 
    YEAR(order_date) as year,
    MONTH(order_date) as month,
    COUNT(*) as order_count,
    SUM(amount) as total_sales
FROM orders 
GROUP BY YEAR(order_date), MONTH(order_date)
ORDER BY year, month;

GROUP BY 的重要规则 ​

1. SELECT 中的列规则 ​

sql
-- ✅ 正确：department在GROUP BY中
SELECT department, COUNT(*)
FROM employees 
GROUP BY department;

-- ❌ 错误：name不在GROUP BY中
SELECT department, name, COUNT(*)  -- name会报错
FROM employees 
GROUP BY department;

-- ✅ 正确：使用聚合函数
SELECT department, COUNT(*), GROUP_CONCAT(name)
FROM employees 
GROUP BY department;

2. 配合 HAVING 使用 ​

sql
-- 找出员工数量超过5人的部门
SELECT department, COUNT(*) as count
FROM employees 
GROUP BY department 
HAVING COUNT(*) > 5;

记忆技巧 ​

  • GROUP BY = 分组 + 聚合
  • 把相同的数据合并成一行
  • 必须配合聚合函数使用
  • SELECT 中的非聚合列必须在 GROUP BY 中

简单来说，GROUP BY 就是**"把相同的东西放在一起，然后计算统计信息"**！
