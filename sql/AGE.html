AGE ​

AGE 是 PostgreSQL 的日期函数，用于计算两个日期之间的间隔并返回为区间。

AGE 函数的基本概念 ​

AGE 计算两个日期之间的差值，返回结果为 INTERVAL 类型。

语法 ​

sql
AGE(timestamp_end, timestamp_start)
-- 返回：两个日期之间的间隔

AGE(timestamp)
-- 返回：从指定日期到当前日期的间隔

常见用法示例 ​

1. 计算年龄（最常用） ​

sql
-- 计算从出生日期到现在的年龄
SELECT 
    name,
    birth_date,
    AGE(birth_date) as age
FROM users;

-- 返回结果示例：
-- name  | birth_date | age
-- ------|------------|------------------
-- 张三  | 1990-05-15 | 34 years 7 mons 5 days

3. 在业务场景中的应用 ​

sql
-- 计算员工的工龄
SELECT 
    employee_id,
    employee_name,
    hire_date,
    AGE(hire_date) as service_years
FROM employees;

-- 计算订单处理时间
SELECT 
    order_id,
    created_at,
    completed_at,
    AGE(completed_at, created_at) as processing_time
FROM orders
WHERE completed_at IS NOT NULL;

-- 计算用户账户注册时长
SELECT 
    user_id,
    registration_date,
    AGE(registration_date) as account_age
FROM users;

4. 提取年龄的具体数字 ​

sql
-- 获取年龄的年份数
SELECT 
    name,
    EXTRACT(YEAR FROM AGE(birth_date)) as age_in_years
FROM users;

-- 获取月份的差值
SELECT 
    order_date,
    EXTRACT(MONTH FROM AGE(order_date)) as months_old
FROM orders;

返回值格式 ​

AGE 返回一个 INTERVAL 类型，格式如：

  • 34 years 7 mons 5 days
  • 2 years 3 months 15 days
  • 6 mons 10 days

与其他方法的对比 ​

sql
-- 方法1: 使用 AGE() - 简单直观
SELECT AGE('1990-05-15') as age1;

-- 方法2: 使用日期相减 - 返回数值
SELECT CURRENT_DATE - '1990-05-15' as days1;

-- 方法3: 使用 EXTRACT - 只能获取年份
SELECT EXTRACT(YEAR FROM CURRENT_DATE) - EXTRACT(YEAR FROM '1990-05-15') as age3;

实用示例 ​

sql
-- 查找年龄在18-65岁之间的用户
SELECT * FROM users
WHERE EXTRACT(YEAR FROM AGE(birth_date)) BETWEEN 18 AND 65;

-- 计算项目的开发周期
SELECT 
    project_name,
    start_date,
    end_date,
    AGE(end_date, start_date) as development_period
FROM projects
WHERE end_date IS NOT NULL;

-- 按年龄排序
SELECT 
    name,
    birth_date,
    EXTRACT(YEAR FROM AGE(birth_date)) as age
FROM users
ORDER BY AGE(birth_date);
