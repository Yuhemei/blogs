Date ​

常见基本用法 ​

js
const date = new Date();
console.log(`output->date.toLocalString()`,date.toLocaleString()) //output->date.toLocalString() 6/6/2024, 7:43:57 PM
console.log(`output->date.toString()`,date.toString())//output->date.toString() Thu Jun 06 2024 19:44:26 GMT+0800 (China Standard Time)
console.log(`output->date`,date)//2024-06-06T11:45:48.823Z
console.log(`output->date.valueOf()`,date.valueOf())//1717674370266 因为valueOf()返回的是时间戳，所以是可以直接用于时间比较

拼写时间字符串 ​

js
const date = new Date();
const year = date.getFullYear();
const month = date.getMonth() + 1;
const day = date.getDate();
const hours = date.getHours();
const minutes = date.getMinutes();
const seconds = date.getSeconds();

const formattedDate = `${year}-${month.toString().padStart(2, '0')}-${day.toString().padStart(2, '0')} ${hours.toString().padStart(2, '0')}:${minutes.toString().padStart(2, '0')}:${seconds.toString().padStart(2, '0')}`;

console.log(formattedDate); // 输出格式为 "YYYY-MM-DD HH:MM:SS" 的字符串
