Promise 基础概念

  1. Promise是一个“类”，通常以实例对象的形式存在。
  2. 应用Promise的异步方法所return的是一个Promise对象。
  3. 一个Promise对象在应用过程中必须是以下三个状态之一(一个代表过程状态，两个代表结果状态)
    • pending 进行中
    • fulfilled 执行完毕
    • rejected 执行失败
  4. 当Promise对象get fulfilled or rejected，We can use the then()
  5. then(),catch()和finnaly()可被链式调用
